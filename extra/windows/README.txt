Description
===========

This directory contains a Windows service which listens for snapshot requests in a TCP port (11321 by default) and makes snapshots on demand.

NSSM - the Non-Sucking Service Manager

http://nssm.cc/

ncat
====

http://nmap.org/ncat/

Install
=======

Copy this directory to c:\Elkarbackup and double click on INSTALL.bat

Uninstall
=========

Double click on UNINSTALL.bat, then you are free to remove the c:\Elkarbackup directory.
