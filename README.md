# Elkarbackup

## What is it?
Elkarbackup is a web interface for the RSnapshot backup software

## Features
- Centralized backups
- Easy-to-use web interface
- Backup Linux/Windows clients
- Instant recovery!
- Pre-scripts & Post-scripts
- Multilanguage (English, Basque, Spanish)
